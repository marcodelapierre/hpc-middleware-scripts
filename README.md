## HPC-middleware-scripts

Scripts to install middleware for HPC.

These scripts have been tested on Ubuntu XX.04, with XX either 18 or 24, case by case.

Current categories:
* `container-engines`
* `container-tools`
* `dev-tools`
* `gpu`
* `job-schedulers`
* `k8s-apps`
* `modules`
* `mpi`
* `package-managers`
* `user-portals`
* `vm-cloud`
* `workflow-engines`
